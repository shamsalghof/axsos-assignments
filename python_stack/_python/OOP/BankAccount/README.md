Bank Account OOP -python

**** Description ****
This project demonstrates Object-Oriented Programming (OOP) in Python by creating a BankAccount class that simulates basic banking operations.

**** Features ****
Create bank accounts
Deposit money
Withdraw money
Display account balance
Method chaining support

**** Concepts Used ****
Classes & Objects
Constructors (__init__)
Instance Attributes
Method Chaining (return self)
Conditional Statements

**** Class Methods ****
deposit(amount)
Adds money to account balance.

withdraw(amount)
Withdraws money if balance is sufficient.
Charges $5 fee if insufficient funds.

yield_interest()
Applies interest if balance is positive.

display_account_info()
Displays current account balance.
