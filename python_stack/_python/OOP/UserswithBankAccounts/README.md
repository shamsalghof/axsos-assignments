# Users With Bank Accounts (OOP Python)

## 📌 Description

This project demonstrates Object-Oriented Programming (OOP) by creating a `User` class that uses a `BankAccount` class to manage financial operations.

Each user has

 Name
 Email
 Bank Account

## 🚀 Features

 Create users
 Deposit money
 Withdraw money
 Transfer money between users
 Display user balance
 Method chaining

## 📂 Project Structure

```
project
│
├── BankAccount
│   └── BankAccount.py
│
└── Users_with_Bank_Accounts
    └── users.py
```

## 🧠 Concepts Used

 Classes & Objects
 Composition (User has BankAccount)
 Method Chaining
 Multiple Objects
 OOP Design

## 💻 Example Usage

```python
first_user.deposit(100).deposit(100).withdraw(50).display_user_balance()

first_user.transfer_money(second_user,30)
```
