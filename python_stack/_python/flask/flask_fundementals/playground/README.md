# 🎮 Flask Play Boxes App

This is a simple Flask project that demonstrates **dynamic routing** and **template rendering** using HTML.

The app generates colored boxes based on URL parameters.

---

## 🚀 Features

- Default route shows 3 red boxes
- Dynamic number of boxes via URL
- Custom box color support
- Uses Flask templates (`render_template`)
- Custom 404 error handler

---

## 📦 Requirements

- Python 3.x
- Flask

Install Flask:

```bash
pip install flask
