# Flask Routes Practice App

This is a simple Flask application created to practice routing, dynamic URLs, and basic error handling in web development.

## 🚀 Overview

The app demonstrates how to:
- Create a Flask application
- Define static and dynamic routes
- Handle URL parameters
- Customize error pages (404)

## 📦 Requirements

- Python 3.x
- Flask

Install Flask using:

```bash
pip install flask
